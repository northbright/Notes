# Combine Multiple PDFs and Images into a PDF on macOS

## Problem
* Need to merge multiple PDFs and images into a single PDF 

## Solution
Use macOS "Quick Action" > "Create PDF" function.

* Select the files

  Please note the order you selected files are the same as created pages in PDF.

* Right click > Quick Action > Create PDF
