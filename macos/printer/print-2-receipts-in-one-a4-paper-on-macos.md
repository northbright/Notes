# Print 2 Receipts in One A4 Paper on macOS

## Problem
* Need to print 2 receipts(PDF) in one A4 paper on macOS
* Also need to print a receipt which size is one half of A4 paper

## Solution
* Open the first receipt(PDF) in preview app
* Go to Edit > Insert > Page from File（页面（来自文件...）)
* Select the second receipt PDF and now the PDF has 2 pages(receipts)
* Press "Command" + "P" to open print settings
  * Orientation: Portrait
  * Turn on "Auto Rotate"（自动旋转） and "Scale to Fit"（自动缩放以适合）
  * Layout > Pages Per Sheet（每张页数）: 2
  * Layout Direction: "Z" or reverted "Z"
