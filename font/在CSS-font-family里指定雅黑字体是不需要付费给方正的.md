# 在CSS-font-family里指定雅黑字体是不需要付费给方正的

* 如果你在图片里使用雅黑字体或者提供雅黑字体下载，需要给方正付费
* 其他用法（比如在 CSS font-family 里指定雅黑字体）是不需要额外付费的，因为雅黑字体是用户电脑自带的
