# Remove a Feishu App

## Problem
* Created an App on [Feishu Open Platform](https://open.feishu.cn/app)
* Need to remove the app
* The "Remove App" link is disabled and it says should disable it first

## Solution
* Visit [Feishu Admin(飞书管理后台)](http://www.feishu.cn/admin) and login
* App Management > Select the App > Config > Click right-top "Disable" button
* Go back to [Feishu Open Platform](https://open.feishu.cn/app) and remove the app
