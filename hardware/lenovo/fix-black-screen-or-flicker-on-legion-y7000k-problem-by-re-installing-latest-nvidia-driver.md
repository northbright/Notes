* Lenovo Legion Y7000K(刃7000K)
  * i9-14900HX
  * NVIDIA GeForce 5060Ti 8GB
* Dell U2723QX 4K monitor
* Display goes black screen frequently(less then 1 min)

Install latest NVIDIA driver(Studio).
