# Panasonic KX-NS300 Can Not Dial Out

## Problem
* It's OK to receive incoming calls
* Can not dial out
* Reboot the device via web does not work

## Root Cause
* Panasonic KX-NS300 ran into problem

## Solution
Do a cold reboot and it works.

* Turn off the device using the I/O switch on back
* Turn on the device again to do a cold reboot
