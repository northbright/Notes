# H3C ER5200 WAN Bandwidth is 100M

## Problem
* ChinaTelecom(Shanghai Telecom) broad service: 500M/100M
* Users reported that download speed was slow

## Root Cause
Use H3C ER5200 as router which WAN bandwidth is only 100M.

## Solution
Use a router which WAN port speed is 1G, 2.5G(e.g. H3C ER5200G2, H3C ER8200G2-X, Huawei S380).
