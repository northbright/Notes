# Windows Memory Management Blue Screen

## Problem
* Got blue screen on Windows 10
* Terminate Code: `MEMORY_MANAGEMENT`
* Need to find out the problem

## Solution
* Run [MemTest64](https://www.techpowerup.com/memtest64/) to see if it's memory issue.
* Can also enter BIOS and run memory test for OEM PC
* Replace RAM if test failed
