# Enter Surface UEFI Settings

## Steps
* Power off Surface
* Keep pressing the "Volume Up" button and press then release the "Power" button
* The screen will show Microsoft / Surface logo, keep pressing the "Volume Up" button
* If it shows UEFI screen, release the "Volume Up" button
