# Print Docs in WeChat on Phone Using EPSON iPrint

## Problem
* EPSON L6168 printer
* It can print photos in iPhone Photos App using AirPrint
* Can not use AirPrint to print docs in WeChat

## Solution
Use EPSON iPrint App to print docs in WeChat.

* Install EPSON iPrint App in AppStore
* Run EPSON iPrint > Select EPSON Printer by IP
* Open doc in WeChat > Click "..." > Open from Other Apps（用其他应用打开）
* Scroll the app list to the most **RIGHT**（拖动 App 列表到最右侧）> More（更多）
  * Warning: If you drop down the **Action MENU（操作菜单）** and select EPSON iPrint, it may report error:
 the file format is not supported（不支持当前文件格式）
* Select EPSON iPrint
