# Download Files from Baidu Net Disk to SMB Share

## Problem
* Windows 10
* Need to download videos on Baidu Net Disk to local SMB share(e.g. `\\server\share`)
* Can not specify the path of SMB share when creating a download task

## Solution
[Add a network location on Windows](https://github.com/northbright/Notes/blob/master/Windows/network/add-a-network-location-on-windows-11.md) and specify the network location in the task.

## References
* [Add a network location on Windows](https://github.com/northbright/Notes/blob/master/Windows/network/add-a-network-location-on-windows-11.md)

# 从百度网盘下载文件到 SMB 共享

## 问题
* Windows 10
* 需要从百度网盘下载文件到 SMB 共享(e.g. `\\server\share`)
* 不能在创建下载任务时指定 SMB 共享的路径

## 解决方法
* [Add a network location on Windows(添加一个网络位置)](https://github.com/northbright/Notes/blob/master/Windows/network/add-a-network-location-on-windows-11.md)，然后在任务中指定该网络位置。
