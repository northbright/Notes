# Install Old Version of FireFox

## Problem
* Firefox 148(Windows x64) shows numbers incorrectly on some page
* 144 version is OK
* Need to install old version of Firefox

## Solution
* Backup Bookmarks
  * Go to Bookmarks > Manage Bookmarks > Backup / Restore icon > Export Bookmarks to HTML

* Uninstall Firefox
  * Go to Control Panel > Programs > Uninstall a Program > Select Firefox > Uninstall

* Download Old Version of Firefox and Install
  * Go to [Directory of other versions and languages](https://ftp.mozilla.org/pub/firefox/releases/)

* Turn Off Automatic Update
  * Go to Settings > General > Firefox Updates > Allow Firefox to Check for updates but let you choose to update them

* Import Bookmarks from HTML file
  * Go to Bookmarks > Manage Bookmarks > Backup / Restore icon > Import Bookmarks from HTML
