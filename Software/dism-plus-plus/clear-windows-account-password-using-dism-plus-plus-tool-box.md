# Clear Windows Account Password using Dism++ Tool Box

## Problem
* Forget Windows account password
* Boot PC from USB disk with [WePE v2.3](https://www.wepe.com.cn/)
* [WePE v2.3](https://www.wepe.com.cn/) has no windows password tool

## Solution
To clear windows account password in [WePE v2.3](https://www.wepe.com.cn/), use the Tool Box in Dism++.

* Run Dism++ > Select Windows installed on local partition(e.g. `c:`) > Tools > Tool Box > Account Management > Select the account > Clear Password

# 使用 Dism++ 的工具箱来清空 Windows 账户的密码

## 问题
* 忘记 Windows 账户的密码
* 从安装 [WePE v2.3](https://www.wepe.com.cn/) 的 U 盘启动电脑
* [WePE v2.3](https://www.wepe.com.cn/) 没有 Windows 密码工具 

## 解决方法
要在[WePE v2.3](https://www.wepe.com.cn/)中清除 Windows 账户密码，可以使用 Dism++ 的工具箱。

* 运行 Dism++ > 选择本地分区的 Windows(e.g. `c:`) > 常用工具 > 工具箱 > 账户管理 > 选择账户 > 清空密码
