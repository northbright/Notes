# Uninstall Microsoft PC Manager

## Problem
* Windows 11 Home
* Need to uninstall Microsoft PC Manager（微软电脑管家）
* Can not find it in Control Panel > Uninstall a program

## Root Cause
* Microsoft PC Manager（微软电脑管家） was installed from Microsoft Store

## Solution
Go to Startup > Installed Apps > Select Microsoft PC Manager > Uninstall
