# 中国电信政企宽带拆机退费流程

* 上海电信政企宽带套餐（6880元/年）拆机
* 电信经理办理后说退费会在办理成功后的 1 个月的账单周期返回到分账序号中，不会直接原路返回到公司银行账户 
* 电信经理目前没有权限可以提现转账到公司银行账户
* 需要到电信营业厅（e.g. 上海长宁区江苏路500号电信大楼）现场办理
* 首先确认分账序号的公司账户（发票抬头）与和电信签合同时的甲方（公司）是否一致
  * 如果在电信翼企购微信小程序改过公司账户（发票抬头），需要改成甲方（公司）才能申请转账
  * 需要让电信经理发邮件联系后台修改公司账户
* 需要的材料
  * 公司委托书（盖公章）：没有电信经理的版本

    ```
    授权委托书
    中国电信股份有限公司上海分公司:
    本单位: XXXX - (统一社会信用代码为: XXXX) ，
    现全权委托本单位员工: XX (身份证号为: XXXX)至贵公司办理 宽带 (手机/固话/宽带)业务的 拆机 (新装/移机/变更)事宜，本单位使用上述业务用于: 公务 。

    本单位对于受托人办理上述事宜时所签署的所有协议、规则、 登记单等文件予以认可，并承担上述业务相应的权利，义务。 

    ________(盖章)
    法定代表人: _________ （签字）
    日期: __年__月__日
    ```

  * 公司营业执照副本（盖公章）
  * 经办人身份证正反复印件（盖公章）
  * 公司的银行开户行，账户以及税号，打印出来即可
